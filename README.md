# Bellcorp Tracker - Backend API

This is the server-side API for the Bellcorp Expense Tracker application.

## Tech Stack

- Node.js & Express.js
- MongoDB (Mongoose)
- JWT Authentication

## Setup

1. Run `npm install`
2. Create a `.env` file with:
    - PORT=5000
    - MONGO_URI=your_mongo_url
    - JWT_SECRET=your_secret
3. Run `npm start`
